## GOALLogAnalyser
Analyses GOAL logs.

### Current Features:
#### Per Agent:
- Module statistics
- Cycle statistics
#### Per Agent Type:
- Module statistics
- Cycle statistics
#### Output formats:
- Text files
- JSON
- HTML & CSS


### How to use:
#### Preperation:
0. In _Eclipse_ > __Window__ > __Preferences__
1. _Goal_ > __Logging__
2. Enable:
    * Write Logs to file
    * The reasoning seperator
    * Include statistics each cycle seperator
    * Entry of a module
    * Exit of a module
3. Run the MAS.
